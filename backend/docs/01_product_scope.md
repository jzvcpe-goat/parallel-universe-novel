# 产品范围定义

## NarrativeOS 不是
- 不是单纯的“AI 续写器”
- 不是随机多结局老虎机
- 不是靠 temperature 撑多样性的文本生成器

## NarrativeOS 是
一个将 **世界状态、角色目标、玩家意图、戏剧推进、承诺账本** 结构化维护，并通过 **搜索 + 批评 + 渲染** 驱动的叙事操作系统。

## MVP 边界

### 要做
- 世界观输入
- 状态驱动的事件搜索
- 玩家意图向量
- Promise Ledger
- route 打分与可解释性
- 多 critic 校验
- 可回放 route

### 暂不做
- 多模态生成
- 大规模社交系统
- 复杂经济系统
- 影视改编工作流
- 完整内容审核平台
- 支付结算系统

## 用户价值

### 玩家
- 不是“看作者给你的三个选项”
- 而是“把自己的命运倾向投射进故事系统”

### 创作者
- 用 world bible 和 creator controls 约束世界灵魂
- 让 AI 成为“剧情推演器”，而不是“抢作者位置的写手”

### 平台
- 拥有可测量、可优化、可运营的路线系统
- 不是一次性内容生产，而是可复用的叙事基础设施
